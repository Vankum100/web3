# Uniswap V3 Liquidity Manager

## Setup Instructions
1. **Install dependencies**:
    ```sh
    npm install
    ```

2. **Compile the contracts**:
    ```sh
    npx hardhat compile
    ```

3. **Deploy the contract**:
    ```sh
    npx hardhat run scripts/deploy.js --network hardhat
    ```

4. **Run tests**:
    ```sh
    npx hardhat test
    ```
